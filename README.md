# PowerBI-Sales-Dashboard  

A Power BI dashboard designed to monitor sales performance and provide actionable insights. This repository highlights report design, data modeling, and visualization techniques using simulated datasets for demonstration purposes.  

## 🎯 Goals  
1. Create a calculated column to compute the sales amount of each transaction.  
2. Build measures for:  
   - Total sales  
   - Total quantity sold  
   - Total number of customers  
3. Develop multiple visualizations to present trends and distributions clearly.

**Key Features:**  
- **Dataset:** Retail sales data (simulated for confidentiality)  
- **Visuals:**  
  - Sales distribution by region  
  - Sales distribution by payment method 
  - Regional and channel breakdown
  - Product category analysis
  - Monthly sales trend 
  - Detailed sales table with region/category breakdown  
- **Techniques Used:** Calculated columns, DAX measures, interactive slicers, cards, pie charts, bar charts, and line charts
